# ドキュメント索引

> 最終更新: 2026-06-11

ローカルAIチャットWebアプリ（Nordic Chat）の仕様・設計ドキュメント一覧。プロジェクト概要はリポジトリ直下の [`../README.md`](../README.md) を参照。

ドキュメントは連番（`01`〜`10`）で管理し、**種別ごと**に下記の3グループへ分類している。新規追加時は次の連番を付与し、本索引の該当グループにも追記する。各ファイル冒頭には「最終更新」を記載する。

## 読み進め順（おすすめ）

`01 要件 → 02 アーキテクチャ → 03 API → 09 エージェント → 05 セキュリティ → 06 テスト → 07 タスク → 10 ロードマップ`。
04・08 は随時参照。初めて環境構築する場合は [`../manuals/setup-guide-macos.md`](../manuals/setup-guide-macos.md) のセットアップ手順から。
